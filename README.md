# nMQ — Hybrid AI Content Ecosystem

Fully automated content production stack built on open-source tools.
Designed to generate, process, and publish educational content at scale.

## Stack
- **Automation:** n8n (self-hosted)
- **Image/Video generation:** ComfyUI, SwarmUI (AMD GPU)
- **Local LLM inference:** LM Studio
- **Infrastructure:** Raspberry Pi 5 (24/7) + AMD GPU workstation (Wake-on-LAN)

## Goals
- Build and document reproducible AI workflows
- Produce tutorial content for TikTok, Instagram, YouTube
- Share everything publicly: workflows, configs, guides
